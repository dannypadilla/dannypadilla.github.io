Gate Detection

Object dection combining preprocessing techniques and machine learning

OpenCV provided preprocessing techniques such as color filtering, thresholding, and contours detection. Results provided regions of interest (ROI) to minimize computation. Features were then extracted using the Histogram of Oriented Gradients(HOG). Training data was provided by the customer (Robsub). Scikit-learn provided the machine learning library. Support Vector Machines(SVM) was used to classify whether the ROI was a gate/bar.
